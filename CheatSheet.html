World Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?

The most essential technologies for the World Wide Web and also those that made it such a great success is the Uniform Resource Identifier (URI) and the HyperText Transfer Protocol (HTTP)

2. How are resources (html pages, images, etc) addressed in the web?

Resources in the web are addressed with an URL

3. What are the general components of a url?

An URL indicates a protocol (http), a hostname (www.example.com) and a file name(index.html)

Web Server

1. What is a web server?

A web server is a computer system that processes requests via HTTP, the basic network protocol used to distribute information on the World Wide Web

2. What does a web server?

A web server stores, process and deliever data to clients

3. How can content be stored on a web server?

Content is stored on their databases. The HTML file lies on the web server which the client can get to see the page

4. How is content in webpages described?

The content from webpages is written in HTML. The client gets the static HTML server and the browser

Web Client

1. What is a web client?

A web client is a software application. It is mostly referred to as a browser

2. What does a web client?

It retrieves, presents and traverses information on the World Wide Web
